# Проверка Telegram-команды /sync

Дата проверки: 2026-07-12 (МСК).

## Назначение

Команда `/sync` строит предварительный план синхронизации и отправляет пользователю только количества действий. Содержимое задач, токен и JSON-ключ в ответ не включаются.

## Безопасность

- Используется область Google `spreadsheets.readonly`.
- Вызов `execute_sync(..., apply=False)` запрещает запись.
- Ошибки Google записываются в локальный журнал, а пользователю отправляется нейтральное сообщение.
- Блокирующий `gspread` выполняется через `asyncio.to_thread`.
- Применение синхронизации из Telegram пока отсутствует, поскольку сначала нужно ограничить его Telegram ID владельца.

## Автоматическая проверка

Проверены варианты ответа:

- источники совпадают;
- найдены будущие изменения;
- обнаружен конфликт.

Прямой вызов той же функции на рабочей таблице вернул:

```text
SQLite → Google: 0
Google → SQLite: 0
Без изменений: 3
Конфликты: 0
```
