# Проверка чтения Google Таблицы через API

Дата проверки: 2026-07-12 (МСК).

## Способ проверки

Команда:

```powershell
.\.venv\Scripts\python.exe tools\check_google_sheet.py
```

Программа использует `gspread==6.2.1` и область доступа `spreadsheets.readonly`. Поэтому во время этой проверки она не может изменять ячейки.

## Результат

- Авторизация сервисного аккаунта: успешно.
- Таблица: `Telegram Tasks - evgeny_team_tasks_2026_bot`.
- Лист: `Задачи`.
- Проверено заголовков: 7.
- Обнаружено строк с задачами: 3.
- Тексты задач и содержимое закрытого ключа в терминал не выводились.

Следующий этап: спроектировать синхронизацию SQLite и Google Таблицы с понятным правилом разрешения конфликтов.
