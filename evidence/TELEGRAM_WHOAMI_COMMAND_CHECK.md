# Проверка Telegram-команды /whoami

Дата проверки: 2026-07-12 (МСК).

## Назначение

Команда `/whoami` возвращает `message.from_user.id` и публичный username. Числовой ID будет использоваться для ограничения команды применения синхронизации.

## Автоматическая проверка

- Ответ содержит числовой ID.
- Username отображается с `@`.
- Отсутствующий username обрабатывается понятной подписью `не задан`.
- `/start` содержит подсказку о новой команде.

## Polling

После изменения `bot.py` прежний процесс проекта остановлен, обновлённый процесс запущен через локальный `.venv`. Подтверждён один polling-процесс `@evgeny_team_tasks_2026_bot`.

Ручная проверка выполнена в 14:28 МСК: бот вернул числовой ID и username владельца. ID добавлен только в локальный `.env`. Скриншот намеренно не копировался в `evidence`, чтобы персональный идентификатор не попал в публичный GitHub.
