# Автоматическое поле «Обновлено (UTC)»

Файл `Code.gs` предназначен для привязанного к Google Таблице Apps Script.

## Что делает сценарий

- реагирует только на ручные изменения листа `Задачи`;
- следит за колонками B «Задача», E «Статус» и F «Категория»;
- записывает текущее время UTC в колонку G;
- поддерживает вставку сразу нескольких строк;
- не изменяет строку заголовков и другие листы;
- не запускается повторно от собственной записи времени.

## Почему не нужен Deploy

Имя `onEdit` зарезервировано Google Apps Script для простого триггера. Сценарий,
привязанный к таблице, запускается автоматически при ручном изменении ячейки.

## Проверка

1. Изменить статус или категорию одной задачи.
2. Убедиться, что в колонке G появилось новое время в формате ISO 8601.
3. Запустить локальный dry-run:

```powershell
.\.venv\Scripts\python.exe tools\sync_google_sheet.py
```

Ожидаемый план для изменённой строки: `PULL_TO_SQLITE=1`.
