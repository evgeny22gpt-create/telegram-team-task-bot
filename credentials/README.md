# Папка ключа сервисного аккаунта

Сюда позже будет помещён скачанный из Google Cloud JSON-ключ:

credentials/google-service-account.json

JSON содержит закрытый криптографический ключ. Его нельзя:

- публиковать на GitHub;
- пересылать в Telegram;
- вставлять в исходный код;
- показывать на скриншотах.

Правило credentials/*.json в .gitignore исключает такие файлы из Git.
Сам README.md остаётся в проекте и объясняет преподавателю безопасную структуру.
